TECNOLOGÍAS DOMINADAS

HTML

Lenguaje de etiquetas para la creación de páginas web, utilizado en el mayor porcentaje de sitios web por su versatilidad y sencillez a la hora de crear páginas web estáticas y de rápida lectura.

CSS

Lenguaje útil para modificar etiquetas realizadas en HTML, se usa mayormente en la colorización y organización de las páginas web, domino este lenguaje en su gran mayoría.

JavaScript

Este es un lenguaje de programación que se pueda usar tanto en la web como en scripts funcionales, entre los lenguajes más demandados hoy en día, domino gran parte de este.

Vue.js

Framework de JavaScript que permite crear componentes reactivos en frontend, flexibilizando la creación de recursos desde backend para aplicaciones y páginas web.

Node.js

Entorno de ejecución del lado del servidor para JavaScript que permite el manejo interno de servidores, APIs, bases de datos y demás funciones de utilidad en el desarrollo web y backend.

GNU/Linux

Kernel y sistema operativo, caracterizado por la versatilidad en su línea de comandos, permite un trabajo más ágil y eficiente a la hora de implementarlo para ejecutar diferentes servicios

MariaDB

Base de datos del lado del servidor que permite una fácil gestión de datos, la integración con diferentes plataformas y comúnmente con páginas, aplicaciones web y demás servicios útiles para la información.
