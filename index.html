TECNOLOGÍAS DOMINADAS

HTML

Lenguaje de etiquetas para la creación de páginas web, utilizado en el mayor porcentaje de sitios web por su versatilidad y sencillez, domino gran parte del lenguaje.

CSS

Lenguaje útil para modificar etiquetas realizadas en HTML, se usa mayormente en la colorización y organización de las páginas web, domino este lenguaje en su gran mayoría.

JavaScript

Este es un lenguaje de programación que se pueda usar tanto en la web como en scripts funcionales, entre los lenguajes más demandados hoy en día, domino gran parte de este.

Node.js

Entorno de ejecución del lado del servidor para JavaScript que permite el manejo interno de servidores, APIs, bases de datos y demás funciones de utilidad en el desarrollo web y backend.
